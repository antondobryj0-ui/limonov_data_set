# Эволюция стиля творчества Эдуарда Лимонова

## Описание проекта

Проект посвящен цифровому анализу произведений Эдуарда Лимонова.  
Главная цель работы — проверить, можно ли с помощью методов компьютерного анализа текста выявить различия в лексике и стиле нескольких произведений автора.

В проекте рассматривается не только частотность слов, но и то, какие тексты оказываются ближе друг к другу по лексико-стилевому профилю.

## Исследовательский вопрос

Как меняется лексико-стилевой профиль текстов Эдуарда Лимонова при сравнении нескольких произведений, написанных в разные периоды его творчества?

Дополнительные вопросы:

- какие слова являются наиболее характерными для каждого текста;
- какие произведения оказываются ближе друг к другу по частотности словоформ;
- можно ли по результатам анализа говорить об изменении авторской манеры.

## Данные

В корпус вошли пять произведений Эдуарда Лимонова:

- Savenko: «Подросток Савенко»;
- epoha: «У нас была великая эпоха»;
- edichka: «Это я — Эдичка»;
- neudachnik: «Дневник неудачника»;
- slugi: «История его слуги».

Тексты были собраны из открытых источников, преобразованы в формат .txt и загружены в Python через ссылки на GitHub.

## Методы

В проекте использованы два основных метода:

1. **TF-IDF**  
   Метод позволяет выделить слова, наиболее характерные для каждого текста по сравнению с другими текстами корпуса.

2. **Стилометрический анализ**  
   Метод используется для сравнения текстов по частотности словоформ. На его основе строится матрица расстояний между текстами, тепловая карта и дендрограмма.

## Основные результаты

Анализ показал, что тексты Лимонова не выстраиваются в простую линейную эволюцию от раннего к позднему стилю.

Наиболее близкими по частотному профилю оказались:

- «Это я — Эдичка»
- «Дневник неудачника»

К ним частично примыкает «История его слуги», хотя этот текст сильнее связан с сюжетной и персонажной лексикой.

«У нас была великая эпоха» занимает промежуточное положение, так как соединяет автобиографический материал с семейно-историческим и военным контекстом.

Самым обособленным текстом оказался «Подросток Савенко», что связано с лексикой детства, семьи, двора и подростковой среды.

## Вывод

Цифровой анализ показал, что внутри прозы Лимонова можно выделить несколько лексико-стилевых зон:

- исповедально-личную
- сюжетно-персонажную
- семейно-историческую
- подростково-автобиографическую

Таким образом, различия между текстами связаны не только с хронологией, но и с жанром, тематикой и типом автобиографического материала.

## Ограничения

Результаты следует рассматривать как предварительные, потому что:

- корпус включает только пять текстов
- произведения различаются по объему
- жанровые особенности могут влиять на частотные показатели
- цифровые методы показывают лексические различия, но не заменяют литературоведческую интерпретацию
