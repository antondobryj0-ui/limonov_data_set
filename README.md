Исследовательский вопрос

Цель проекта - проверить, можно ли с помощью методов компьютерного анализа текста выявить изменения в стиле и лексике произведений Эдуарда Лимонова разных периодов.

Основной исследовательский вопрос:

Как меняется лексико-стилевой профиль текстов Эдуарда Лимонова при сравнении нескольких произведений, написанных в разные периоды его творчества?

Этот вопрос можно проверить на данных, потому что тексты представлены в формате «.txt», а значит, их можно обработать программно: посчитать частотность слов, выделить ключевые слова и сравнить тексты между собой по частотным признакам.

Дополнительные вопросы:

1. Какие слова являются наиболее характерными для каждого текста?
2. Какие тексты оказываются ближе друг к другу по частотности словоформ?
3. Можно ли по результатам анализа говорить об изменении авторской манеры?

Предыдущие исследования

Творчество Эдуарда Лимонова часто рассматривается в связи с автобиографичностью, образом автора, эмигрантским опытом и соединением литературы с политическим жестом. Особенно часто в этом контексте упоминается роман «Это я - Эдичка», который воспринимается как важный текст русской автобиографической прозы и как произведение, где литературный герой тесно связан с публичным образом автора. 

Исследователи также обращают внимание на литературные взгляды Лимонова, его отношение к русской литературе XX–XXI веков, андеграунду и понятию «жестокого реализма». Это важно для нашего проекта, потому что стиль Лимонова связан не только с темами, но и с особой авторской позицией: резкостью, провокативностью, автобиографичностью и сознательным разрушением привычных литературных норм. 

Методологически проект опирается на подходы цифровой гуманитаристики и текст-майнинга. TF-IDF используется для выделения слов, наиболее значимых для конкретного текста внутри корпуса. Этот метод позволяет увидеть не просто самые частотные слова, а слова, которые отличают один текст от других. 

Для сравнения текстов по стилю используется стилометрический подход. В классической стилометрии важны частотные признаки текста: повторяющиеся словоформы, устойчивые языковые привычки, распределение наиболее частотных слов. Один из известных методов такого анализа - дельта Бёрроуза, предложенная Джоном Бёрроузом для измерения стилистического расстояния между текстами. 

Таким образом, проект соединяет два уровня анализа: тематический, через TF-IDF, и стилометрический, через сравнение частотных профилей текстов.

Метод

В проекте используются два цифровых метода: TF-IDF и стилометрическое сравнение текстов.

Первый метод - TF-IDF. Он позволяет выделить ключевые слова каждого произведения. Обычная частотность показывает, какие слова встречаются чаще всего, но в таких списках часто оказываются слишком общие слова. TF-IDF работает точнее: он повышает вес слов, которые часто встречаются в одном тексте, но редко встречаются в других текстах корпуса. Поэтому этот метод подходит для поиска тематических и смысловых маркеров отдельных произведений.

Второй метод - стилометрический анализ. Для него тексты сравниваются по частотности словоформ. В отличие от TF-IDF, здесь важно не только содержание, но и повторяющиеся речевые привычки. На основе частотных профилей строится матрица расстояний между текстами: чем меньше расстояние, тем ближе тексты друг к другу; чем больше расстояние, тем сильнее они различаются.

Дополнительно используются визуализации: тепловая карта расстояний и дендрограмма. Тепловая карта показывает степень близости текстов друг к другу, а дендрограмма помогает увидеть, какие произведения группируются вместе.

Ход исследования

Сначала тексты были приведены к единому формату .txt. Затем они были загружены в Python через ссылки на GitHub. После этого была проведена предварительная обработка: удаление лишних символов, приведение текста к нижнему регистру, токенизация, лемматизация и удаление стоп-слов для TF-IDF.

На следующем этапе был рассчитан TF-IDF. Для каждого текста были выделены слова с наибольшим весом. Это позволило определить, какие лексические маркеры сильнее всего отличают одно произведение от других.

Затем была проведена стилометрическая часть анализа. Для нее тексты были очищены, но не лемматизированы, потому что в стилометрии важны именно словоформы. После этого была построена таблица частотности наиболее частотных словоформ, рассчитаны расстояния между текстами и создана визуализация.

Ограничения и перспективы

У проекта есть несколько ограничений. Во-первых, корпус текстов ограничен: анализируется только несколько произведений Лимонова, поэтому выводы нельзя автоматически распространять на всё его творчество. Во-вторых, разные тексты могут отличаться по объему, жанру и времени написания, а это влияет на результаты. Например, роман, дневниковая проза и публицистический текст могут различаться не только из-за эволюции стиля, но и из-за жанровых особенностей.

Еще одно ограничение связано с обработкой текста. Если из анализа удаляются стоп-слова, результат становится удобнее для интерпретации, но менее строгим с точки зрения классической стилометрии. Если стоп-слова оставить, анализ будет ближе к авторской атрибуции, но часть результатов будет выглядеть менее понятной для содержательного литературного анализа.

В дальнейшем проект можно расширить. Во-первых, можно добавить больше произведений Лимонова и разделить их по периодам: ранняя эмигрантская проза, постсоветские тексты, поздняя публицистика. Во-вторых, можно отдельно сравнить художественные тексты и публицистику. В-третьих, можно добавить другие методы: анализ длины предложений, частотность местоимений, n-граммы, тематическое моделирование или сравнение Лимонова с другими авторами русского андеграунда и эмигрантской литературы.

Главная перспектива проекта заключается в создании более широкой карты эволюции авторского письма Лимонова.
